# Задача 1 
Решить задачу со следущюими данными:
$x_{1}+2x_{2}\leq 10$
$x_{1}+2x_{2}\geq 2$
$2x_{1}+x_{2}\leq 10$
$x_{1},x_{2}>0$
Целевая функция: $F(x)=x_{1}+x_{2}\to max$

![[Pasted image 20230220183949.png]]

Ответ: $F(x)=6 \frac{2}{3}$
